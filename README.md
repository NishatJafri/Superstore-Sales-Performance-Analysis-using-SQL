# Superstore-Sales-Performance-Analysis-using-SQL
Analyzed Superstore sales data using SQL to explore sales trends, customer behavior, regional performance, and profitability. This project focuses on finding meaningful insights such as top-performing products, high-value customers, and factors affecting business growth through data-driven analysis.
## Project Overview
This project analyzes Superstore sales data using SQL to identify business insights related to sales performance, customer behavior, profit trends, and regional performance.

## Objectives
- Analyze overall sales and profit trends
- Identify top-performing products and categories
- Find high-value customers
- Discover loss-making products
- Study the impact of discounts on profit

## Tools Used
- MySQL
- MySQL Workbench
- SQL

## Dataset
- Superstore Dataset

## Key Analysis
- Total Sales Analysis
- Category-wise Performance
- Region-wise Analysis
- Customer Analysis
- Profit and Discount Analysis
